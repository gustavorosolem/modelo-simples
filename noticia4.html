Anunciado Code Geass: Boukoku no Akito

Code Geass Gaiden: Boukoku no Akito novo filme do grupo CLAMP, conta uma história diferente da apresentada no anime, sendo considerada como uma nova fase de Code Geass.

Code Geass Boukoku no Akito 204x300 Trailer de Code Geass Gaiden Boukoku no AkitoO trailer de Code Geass Gaiden: Boukoku no Akito foi transmitido num evento do estúdio Sunrise e pode ser visto no youtube. Durante o evento foi anunciado que o filme será visto nos cinemas japoneses durante o verão.

Code Geass Gaiden: Boukoku no Akito se passa no continente europeu, no ano de 2017, um pouco antes da segunda temporada do anime, Code Geass: Lelouch of the Rebellion R2.

A animação retrata uma pequena tropa de Nightmares formada por adolescentes do Eleven (o antigo Japão), entrando no campo de batalha europeu para realizar uma missão quase suicida de resgatar um aliado, com chance de sucesso de apenas 5%.

O site oficial do projeto já está disponível, entretanto pouco foi revelado.

Serão quatro OVAS com duração de 50 minutos cada e serão lançadas sempre com uma diferença de 3 meses entre os cinemas e o lançamento dos Blu-ray.

A primeira parte estreia em 4 de Agosto nos cinemas Japoneses.
